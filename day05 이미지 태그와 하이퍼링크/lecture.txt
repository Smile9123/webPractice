이미지 태그 : 이미지를 나타내 줄 수 있는 태그 / 단독태그 <img />\ 글자와 같이 나열되는
성격, 이미지의 단락을 구분 지을 때 "br"

이미지의 확장자(대표)
- jpg : 가장 대표적인 이미지 표현방식, 이미지의 색상을 가장 잘 표현할 수 있는 확장자
- png : 이미지의 색상을 유지하며 투명한 배경을 나타낼 수 있는 확장자
- gif : 투명한 배경이 가능하나, 색상이 256개의 표현만 가능(화질이 떨어져 보임 움짤)

이미지 태그 사용법 : 경로 >> 폴더안 >> 작성하는 문서의 기점으로부터

figcaption : 요소에 캡션을 넣을 떄 사용하는 태그로 figure안에서 사용, 제일 처음이나
마지막에 배치

img 태그 주소 : 검색했을 때 이미지의 주소 복사 후 사용할 수 있으나 저작권에 유의!

a태그 : 웹페이지를 연결하거나 이동할 때 사용하는 태그(메뉴표현할 때 가장 많이 사용)

a태그 특징
- 글자와 같이 나열되는 성격 
- 링크연결 시 기본 스타일(글자색 파란색, 밑줄, 손모양의 커서), 브라우저내에서 바로 전환
- 인터넷 링크 작성 시 http 필수

a태그 임시링크
- #

a태그 메일 전화 문자
- 메일 : mailto
- 전화 : tel
- 문자 : sns

a태그 새창 전환
- target : 링크된 문서를 클릭했을 때 문서가 열릴 위치를 지정
- _self : 기본 값 / 현재의 브라우저에서 전환
- _blank : 새창전환 / 새로운 도메인 창을 열 때
