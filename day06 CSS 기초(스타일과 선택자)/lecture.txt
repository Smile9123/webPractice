-전체 선택자 : html내에 등장하는 모든 형태를 선택하는 선택자(초기화, 기본 설정 값)

-태그 선택자 : h1, h2, p, a, img .../html내에 등장하는 특정한 태그를 선택하는 선택자

-아이디, 클래스 선택자 :  특정한 이름을 부여하여 선택하는 선택로, html화면 내에서 하나의
                        이름으로 여러 번 사용 가능. 주로 작은 요소, 작은 다락, 변화가 
                        잦을 스타일

-외부 스타일시트 css방법 : ex)<link rel="stylesheet" href="css/style.css" />
