볼드체를 표현되어지는 태그들

01. strong태그: bold태그 >>> 경고 주의사항과 같이 중요한 내용들을 강조할 때

02. b태그: bold태그 >>> 글자를 굵게 표현하고 싶을 떄

03. em태그: 이탤릭체 태그 >>> 주위 텍스트에 비해서 강조된 부분이나 저자의 생각

04. i태그: 이탤릭체 태그 >>> 이탤릭체를 표현하고 싶을 때 

05. q태그: 인용표현 태그 >>> 인용할 문구를 표현, 따옴표가 나옴! 글자와 같이 나열되어지며 
           짧은 본문의 인용을 표현할 때 사용

06. mark태그: 형광팬 효과 태그 >>> 중요한 부분을 강조할 때~ 사용

07. span태그: 글자를 꾸밀 때 사용태그 >>> 글자를 꾸밀 때 사용 style, 
              css와 함께 주로 사용
